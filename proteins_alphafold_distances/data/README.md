# Data

## Files

### `1ubq.pdb`
Protein Data Bank file for **Ubiquitin** (PDB ID: 1UBQ), a small 76-residue regulatory protein found in all eukaryotic cells. The structure was refined at 1.8 Angstroms resolution. This file contains atomic coordinates used for geometric distance analysis.

The file is fetched directly from [RCSB PDB](https://www.rcsb.org/structure/1UBQ).

## Regeneration

To re-download the PDB file, run from the project root:

```bash
python src/data_fetcher.py
```

This fetches the file from `https://files.rcsb.org/download/1ubq.pdb`. An internet connection is required.
